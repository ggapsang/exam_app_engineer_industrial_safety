# 산업안전기사 기출 모바일 앱 — 기획안 v2.0

**문서 버전**: v2.0 (2026-06 작성) **대상 플랫폼**: Android 1차 / iOS 차순위 **작업 위탁 방식**: 블랙박스 요구사항 (개발 자율, 인수 기준만 명시)

---

## 1\. 제품 정의

### 1.1 한 줄 정의

산업안전기사 필기 5개년 기출(2017\~2021) **1,800문항·해설·이미지 전부 오프라인 임베드**된 학습 앱. 회차별·과목별·랜덤 출제 \+ 오답노트 \+ 학습 통계.

### 1.2 데이터 자산 (현황 — 인수 시점 기준)

| 항목 | 수치 |
| :---- | :---- |
| 회차 | 15회 (2017·1·2·3 \~ 2021·1·2·3) |
| 문항 | 1,800 (회차당 120\) |
| 보기 | 7,200 (문항당 4\) |
| 정답 | 1,800 (100%) |
| 해설 | 1,800 (100%, AI 생성 \+ 직접 보강 혼합) |
| 그림 이미지 | 204장 (159문항에 매핑, 평균 1.28장/문항) |
| 과목 균형 | 6과목 × 각 300문항 |
| 정답 분포 | 답1:25.1%, 답2:25.3%, 답3:24.6%, 답4:25.0% (균형) |
| DB 크기 | 3.88 MB (SQLite) |
| 이미지 폴더 | 535 KB (GIF) |

### 1.3 타깃

- 산업안전기사 필기 응시자 (재직자 비중 高 — 출퇴근·점심 자투리 시간)  
- 종이 문제집 휴대 불편 \+ 회사 안에서 풀고 싶은 사람  
- 약점 단원 자동 식별·반복 학습 원하는 학습자

### 1.4 핵심 가치 제안

1. **완전 오프라인** — DB·해설·이미지 전부 앱 안에 임베드. 네트워크 불요  
2. **자동 약점 진단** — 풀이 통계로 취약 과목·단원 식별 → 맞춤 출제  
3. **실전 모의고사** — 120문항·2시간 30분 타이머, 60점 합격선 시각화  
4. **그림 문제 지원** — FT도·연쇄도·표 등 이미지 정상 표시

## 3\. 기능 요구사항 (블랙박스)

### 3.1 학습 모드 (5개)

#### F1. 회차별 풀이 (Past Exam Mode)

- **입력**: 회차 선택 (드롭다운 또는 그리드 15개)  
- **출력**: 해당 회차 120문항 순차 출제  
- **동작**:  
  - 옵션 a) 즉시 채점 (선택→정오 표시→다음)  
  - 옵션 b) 일괄 채점 (120문항 풀이 종료 후)  
- **저장**: user\_attempts에 (question\_id, selected\_no, is\_correct, elapsed\_ms)

#### F2. 과목별 풀이 (Subject Mode)

- **입력**: 6과목 중 선택  
- **부가옵션**: 출제 수(20/50/100/300), 정렬(랜덤/연도순)  
- **출력**: 해당 과목 300문항 풀에서 추출

#### F3. 모의고사 (Mock Exam Mode)

- **입력**: "모의고사 시작" 버튼  
- **출력**: 6과목 × 20문항 \= 120문항 자동 조합  
- **제약**:  
  - 동일 회차 비중 ≤ 20%  
  - 5회차 이내 풀이 이력 있는 문항 가중치 ↓  
- **타이머**: 2시간 30분 카운트다운, 자동 제출  
- **결과**: 총점·과목별 점수·합격선(60%) 시각화

#### F4. 오답노트 (Wrong Note Mode)

- **입력**: "오답노트" 진입  
- **출력**: 오답·미응시 문항 목록(과목별 그룹 \+ 즐겨찾기 우선)  
- **부가옵션**:  
  - 필터: "한 번 이상 틀린 것" / "두 번 이상 틀린 것" / "최근 7일"  
  - 정렬: 오답 횟수↓, 과목 순, 회차 순  
- **풀이**: 클릭 시 단건 학습 카드로 진입 (해설 펼침)

#### F5. 즐겨찾기 (Favorite Mode)

- 별표 토글 → favorites 테이블 INSERT/DELETE  
- 별표 문항만 모은 랜덤 출제

### 3.2 학습 보조

#### F6. 해설 카드

- **표시**:  
  - 정답 번호 (강조)  
  - 본문 해설 (Markdown 일부 — 굵게·줄바꿈·리스트)  
  - 참고 법령/공식 (있을 때)  
  - 즐겨찾기 토글 ☆  
  - 메모 작성 영역  
- **이미지 처리**:  
  - `question_images.role='stem'` → 본문 상단에 이미지 렌더  
  - `role='choice1~4'` → 해당 보기 위치에 이미지 렌더  
  - 이미지는 `assets/images/{path}`에서 로드

#### F7. 키워드 검색

- 검색 대상: questions.stem, choices.body, explanations.body  
- OR 조건, 한글 부분일치  
- 결과 카드: 회차·문항번호 \+ 미리보기 30자

#### F8. 학습 통계 대시보드

- **표시 항목**:  
  - 누적 풀이 수 / 정답 수 / 정답률  
  - 과목별 정답률 (막대 그래프 6칸)  
  - 약점 단원 Top 5 (단원 매핑이 없으면 과목 단위 표시)  
  - 최근 30일 일별 학습 시간 (라인 차트)  
  - 학습 지속일 (streak)  
- **갱신**: 매 풀이 종료 시 자동

### 3.3 UX

| 기능 | 요구사항 |
| :---- | :---- |
| F9. 다크 모드 | 시스템 연동 \+ 수동 토글 |
| F10. 폰트 크기 | 3단계 (작게/보통/크게) |

### 3.4 데이터 관리

| 기능 | 요구사항 |
| :---- | :---- |
| F14. 학습 기록 백업 | JSON 파일로 외부 저장소에 저장 |

## 7\. 개발 외주 인수 조건

### 7.1 개발사 자율

- 언어/프레임워크 자유 (Kotlin \+ Compose 권장, Flutter도 허용)  
- 아키텍처 자유 (MVVM/Clean/MVI 등)  
- 라이브러리 자유

### 7.2 필수 인수 조건

1. **DB·이미지 무변경**: `industrial_safety_engineer.db`와 `images/` 폴더를 그대로 임베드, 스키마 변경 없음 (앱 추가 테이블은 별개)  
2. **오프라인 동작**: 비행기 모드에서 F1\~F12 전부 동작  
3. **이미지 정합성**: question\_images에 매핑된 204장 모두 정상 표시  
4. **수동 테스트 시나리오 통과**: 회차별, 과목별, 모의고사, 오답노트 각각 한 사이클 완주  
5. **APK 빌드**: Play Console 내부 테스트 트랙 업로드 가능한 서명 APK  
6. **DB 갱신 절차 문서화**: 추후 회차 추가/오답 정정 시 DB 교체 방법

### 7.3 비인수 사유

- 화면 전환 200ms 초과  
- 그림 문제에서 이미지가 안 보이는 경우  
- 모의고사 출제에 동일 회차가 25문항 이상 포함되는 경우  
- 학습 기록이 앱 종료 후 사라지는 경우
